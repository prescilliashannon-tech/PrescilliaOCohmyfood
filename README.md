# Prescillia OC Ohmyfood

Ce projet consiste à améliorer l’interface mobile-first du site Ohmyfood, en intégrant une maquette, en utilisant Sass pour le CSS et en ajoutant des animations.

Ce dépôt contient :  
- La page d’accueil (index.html)  
- Les fichiers CSS et Sass  
- Les images nécessaires au site  

Ce projet est réalisé dans le cadre de la formation OpenClassrooms.

_variables.scss : Toutes les variables (couleurs, polices, espacements).
_mixins.scss : Mixins et fonctions réutilisables.
_base.scss : Styles globaux et reset (ex : margin, padding, body, titres).
_layout.scss : Styles liés à la structure générale du site (grilles, conteneurs, flexbox).
_components.scss : Styles spécifiques aux composants réutilisables (boutons, formulaires, cartes).
_pages.scss : Styles particuliers pour certaines pages (accueil, restaurant, contact).
